家庭网络升级小记-自己动手更换电信光猫
发表于2020-02-28|更新于2020-08-23|其他
|字数总计:1.2k|阅读时长:3分钟|阅读量:

近来，受疫情的影响啊，各大运营商宽带提速的活动是一个接着一个，这让经常需要大带宽来下载视频的我心动不已啊。但奈何家里面的光猫还是电信N年前装宽带送的百兆猫，这速度提的再高也没用啊，还是只能跑出100兆宽带的速度。
后面上电信商城看了一下，不看不知道、一看吓一跳，一个单口千兆的猫电信硬生生的卖到了299，WDNMD。有这钱，我重新装个宽带白拿一个千兆猫它不香吗。
无奈之下，只能选择自己动手更换光猫，挑了许久，最终选定了华为8245C2，和华为8245H一样的配置，网口全千兆，EPON\GPON双模自适应三网通用。只是华为在软件上进行了屏蔽，8245C2只有前两个网口是千兆并且只支持中国电信EPON线路，但是，我们只需要telnet进去，改一下型号即可恢复成8245H。在转转上面花了60买到，卖家还送了3个路由器，四舍五入一下，简直白给有木有。比起电信那个更是觉得赚翻了。

心念念的等了3天，快递终于到了，迫不及待的去取了快递，拆开一看，三个路由一个猫，整整齐齐的躺在里面。心里面不由得说了一句：良心卖家

光猫既然到手了，前面已经说了，要先把8245C2给刷成8245H，不然家里面的GPON线路用不了。话不多说，直接开搞。

  1. 第一步我们需要先进入老猫的后台，记下它的loid以及其他连接的数值（Other是IPTV看电视用的，INTERENT是上网用的），换了光猫后，只需要照着老猫上面的填就可以了。语音和固话要麻烦一点，不过我也不用，就没弄。

  2. 第二步呢，我们需要给新光猫开启telnet，不然什么操作都做不了。看了下网上8245C2开启telnet改8245H的教程，先通过超级密码(很关键 必须要超级密码)登录电信界面，然后用web developer toolbar插件打开隐藏元素，勾选使能LAN侧PC通过TELNET访问ONT就可以开启光猫的telnet了。

    但是我这个猫买来就已经刷成了8245H，还不是原厂界面，通过超级密码登录进去，使用插件打开了隐藏元素也找不到这个选项。没办法，只能想其他办法了。

    最后我是用华为ONT组播工具开启telnet的，这个是华为售后专门用来开telnet的工具，大部分的猫都支持。如有需要的，请点击这里下载。首先打开软件，点击维修使能，选择连接光猫的那张网卡，点击启动，等到光猫网口灯全亮的时候就可以点停止了，然后重启光猫。不出意外的话，应该能够成功telnet上去了。

  3. telnet上去以后，输入账号root密码admin。再输入su回车，在challenge后面有一串字符，把它复制到华为光猫su密码计算器里面，点击计算，然后复制得出的值到CMD里面粘贴并回车，就能进入超级用户模式了。再输入shell进入超级用户的wap模式。因为我的猫已经是8245H了，我只需要输入恢复原厂和改双模的命令就行了。改双模的命令要在超级用户模式下输入，不然会报错，全部执行后输入reboot重启光猫。

  4. 重启完成后进入192.168.100.1这是华为原厂界面的管理地址。输入账号telecomadmin密码admintelecom登录。在上面的菜单栏找到系统工具，点击里面的ONT认证，输入你的loid密码为空，点击应用。应用之后再点击菜单栏的状态，查看业务的开通状态出现OLT业务配置成功就说明可以上网了，那个ACS注册不用管，那是电信的远程管理。

    OLT业务配置成功后，点击WAN在里面新建连接，填入和你老猫一样的参数就可以了。（部分地区可能会绑定MAC地址，修改MAC地址网上也有）

这次更换光猫呢到现在就完了，家里面的200兆宽带终于能跑满了。更换过程总得来说还是蛮简单的，就是记几个参数敲几个命令。后面再试一下配合Openwt固件看能不能实现多拨。
